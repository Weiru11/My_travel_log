# My Australia Travel Log

This website is a personal travel diary about places I visited in Australia.  
Users can explore destinations through an interactive map and read travel memories by activities, food, and views.

## Website Pages

- Home page: `index.html`
- Sydney: `sydney.html`
- Melbourne: `melbourne.html`
- Brisbane: `brisbane.html`
- Whitsunday: `whitsunday.html`

## Features

- Fixed navigation bar
- Places dropdown menu
- Interactive Australia map
- City hero images
- Image sliders with matching descriptions
- Responsive layout for different screen sizes

## Technologies Used

- HTML
- CSS
- JavaScript
